# 📝 Todo List App

Aplicación de gestión de tareas construida con **Angular 17+** Permite crear usuarios, agregar, editar, completar y eliminar tareas consumiendo una API REST externa.

---

## ⚙️ Requisitos

| Herramienta | Versión usada |
|---|---|
| Node.js | 22.22.2 |
| npm | 10.9.7 |
| Angular CLI | 21.2.9 |

---

## 🚀 Correr en local

```bash
# 1. Clonar el repositorio
git clone https://github.com/MarioDR25/todolist-angular-MarioD.git
cd TU_REPOSITORIO

# 2. Instalar dependencias
npm install

# 3. Correr la app
ng serve
```

Abrir en el navegador: `http://localhost:4200`
